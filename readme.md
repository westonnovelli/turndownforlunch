#Turn Down For Lunch

Blackbaud - Off the Grid Summer '14

### How to run!
1. bundle install
2. rake db:migrate
3. rails server
4. navigate to localhost:3000
5. enjoy!


### API

    Suggestion:
	
	- string location
	- string departure time
	- integer votes

-----
    
    User:
	
	- string name
	- integer suggestion
